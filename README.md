# Gestion Chantier BTP

## Description

Application de gestion et de suivi de chantier BTP en temps réel.

L'objectif est de permettre aux différents acteurs d'un chantier de suivre l'avancement des travaux, les tâches, les ressources, les documents et les informations importantes depuis une interface centralisée.

## Objectifs du projet

- Suivi de l'avancement des chantiers
- Gestion des utilisateurs et des rôles
- Suivi des tâches et des interventions
- Gestion documentaire
- Suivi des ressources
- Visualisation de l'état des chantiers en temps réel

## Architecture technique

Le projet est organisé autour d'une architecture moderne :

- Backend : Spring Boot (Java)
- Frontend : React
- Base de données : PostgreSQL
- Conteneurisation : Docker
- Environnement de développement : Dev Container / GitHub Codespaces
